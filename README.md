# Co & Co Law Firm Website (Updated with Your Logo)

This is a clean, minimal boutique-law-firm website that works as a normal static website.
You do not need GitHub to use it.

## Included files

- `index.html`
- `styles.css`
- `script.js`
- `assets/logo.png`

## How to preview

Open `index.html` in your browser.

## How to publish without GitHub

### Option 1: Netlify
1. Sign in to Netlify.
2. Drag the full website folder into the deployment area.
3. Connect your Porkbun domain in Netlify domain settings.

### Option 2: Vercel
1. Create a new static project.
2. Upload these files.
3. Connect your Porkbun domain in Vercel.

### Option 3: Traditional hosting
1. Open your hosting control panel.
2. Upload all files into `public_html`.
3. Make sure `index.html` is in the root.
